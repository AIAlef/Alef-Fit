### Added
- M0 scaffold: project structure, architecture & planning docs (`docs/`), `CLAUDE.md`.
- Runnable Flutter app shell: 5 bottom-nav tabs (Exercise / Discipline / Program / Retro / Setting), Material 3 theme, go_router `StatefulShellRoute`, EN/TH localization setup.
- Drift database with core Exercise-library tables and migration strategy.
- Detailed Settings spec (`docs/SETTINGS_SPEC.md`) and design-system plan (`docs/DESIGN_SYSTEM.md`).
- CE (Gregorian) set as the calendar standard (BE optional); data model extended with folders, tags, card backgrounds, and to-do alert defaults.
- Exercise categories finalized (10, priority-ordered): Chest, Back, Leg, Shoulder, Bicep, Triceps, Abs, Compound, Functional, Stretching. Calf folds into Leg; forearm splits flexor→Bicep / extensor→Triceps.
